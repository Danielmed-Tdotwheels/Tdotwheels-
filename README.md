# Tdot Wheels Shopify Theme

This repository contains the Shopify theme for Tdot Wheels, exported on April 1, 2026.

## Structure
- `assets/`: Theme assets (images, fonts, shared JS/CSS).
- `config/`: Theme settings and translation files.
- `layout/`: Key layout files (theme.liquid).
- `locales/`: Translation files.
- `sections/`: Reusable theme sections.
- `snippets/`: Reusable snippets of Liquid code.
- `templates/`: Page templates (product, collection, etc.).

## Development
To work on this theme, it is recommended to use the [Shopify CLI](https://shopify.dev/docs/themes/tools/cli).
